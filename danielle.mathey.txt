1.If you were to describe semantic HTML to the next cohort of students, what would you say?

2.Name two big differences between display: block; and display: inline;.

With display inline you can set a width and height on an element. With display block you can have elements display next to other elements.

3.What are the 4 areas of the box model?

Content, border, padding, margin.

4.While using flexbox, what axis does the following property work on: align-items: center?

5.Explain why git is valuable to a team of developers.

When working on a project with multiple developers using Git you are able to see everyone else's code and you are able to work on different pieces of the project at the same time and see where mistakes were made and be able to figure out how to fix them or revert back with your own code and see if that fixes what's broken.
